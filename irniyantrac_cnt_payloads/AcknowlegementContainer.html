7. Acknowledgement Container

{
  "op": 1,
  "to": "<DEVICE-AE-ID>",
  "fr": "<DEVICE-AE-ID>",
  "rqi": "<unique-Request-Identifier>",
  "ty": 3,
  "pc": {
    "m2m:cnt": {
      "rn": "ACKCNT"
    }
  },
  "rvi": "3"
}
  • op (Operation): 1 signifies a create operation.
  • to: The target resource, represented by <DEVICE-AE-ID>.
  • fr: The originator of the request, represented by <DEVICE-AE-ID>.
  • rqi: A unique request identifier.
  • ty (Resource Type): 3 represents a container resource.
  • pc (Payload Content): Contains the resource-specific data. Here, a container with the resource name ACKCNT is being created.
  • rvi (Release Version): Specifies the version of the oneM2M protocol being used, here "3".
